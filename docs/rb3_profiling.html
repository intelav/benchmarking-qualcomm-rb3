Profiling Tools on RB3¶

benchmark_model (TFLite Tool)¶

Purpose¶

  • Measures TFLite CPU inference
  • Uses XNNPACK if enabled

What It Uses¶

  • ARM CPU
  • NEON SIMD
  • XNNPACK kernels

What It Does NOT Use¶

  • DSP
  • SNPE
  • QNN
  • AIP
