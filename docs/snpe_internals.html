SNPE on Host (Model Conversion)¶

  • Download Model from http://download.tensorflow.org/models/mobilenet_v1_2018_08_02/mobilenet_v1_1.0_224_quant.tgz
  • Download https://github.com/sourcecode369/tensorflow-1/tree/master/tensorflow/lite/examples/label_image/testdata/grace_hopper.bmp
curl http://download.tensorflow.org/models/mobilenet_v1_2018_08_02/mobilenet_v1_1.0_224_quant.tgz --output ./mobilenet_v1_1.0_224_quant.tgz
tar -xvf mobilenet_v1_1.0_224_quant.tgz

Model Conversion to Dlc Format¶

snpe-tflite-to-dlc   --input_network mobilenet_v1_1.0_224_quant.tflite   --output_path mobilenet_v1_1.0_224_quant.dlc

Inspect Dlc Model¶

snpe-dlc-info -i mobilenet_v1_1.0_224_quant.dlc | more

copy dlc model to RB3

Create Input¶

copy grace_hopper.raw to RB3
