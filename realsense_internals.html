Intel RealSense Internals¶

Intel RealSense cameras are depth-sensing systems.

Components¶

  • RGB camera
  • Infrared cameras
  • Depth processor

Goal¶

  • Compute per-pixel depth
  • Align depth with color data

Used In¶

  • Robotics
  • SLAM
  • 3D perception

Stereo Depth Principle¶

Depth is computed using stereo vision.

Process¶

  • Capture left and right IR images
  • Match corresponding pixels
  • Compute disparity
  • Convert disparity to depth

Depth Accuracy Depends On¶

  • Baseline distance
  • Calibration quality
  • Lighting conditions

RealSense Data Streams¶

RealSense cameras output multiple synchronized streams.

Common Streams¶

  • Color (RGB)
  • Depth (Z16)
  • Infrared (IR)
  • Metadata and confidence

Properties¶

  • Independent resolution and FPS
  • Hardware time synchronization

RealSense Processing Pipeline¶

RealSense performs on-device depth processing.

Pipeline Stages¶

  • Sensor capture
  • ISP processing
  • Stereo matching
  • Depth post-processing

Post-Processing Includes¶

  • Spatial filtering
  • Temporal filtering
  • Hole filling

Result¶

  • Cleaner depth maps
  • Reduced noise
  • Stable perception
